1、let和const命令

[TOC]

¶let和const

¶let

  • let 变量=值
  • 只在当前作用域内有效(可用于for循环) 即{} 这样的代码块
  • 不存在变量提升(提前声明)
  • 存在暂停性死区(在变量声明前不能使用该变量)
  • 不允许重复声明

¶使用var的风险

  • 内层变量可能会覆盖外层变量 (提前声明)
  • 用于循环计数的变量泄露为全局变量

¶es6的块级作用域

  • 只要是{}就是一个块级作用域

  • 可以任意嵌套 内层作用域可以定义外层作用域的同名变量

  • 允许在块级作用域中声明函数(es5前不允许 在严格模式下同样无效)

  • 本质上es6的块级作用域是一个语句 将多个操作封装在一起(操作即作用域中的执行语句) 没有返回值(所有外层无法访问内层)

  • do{} 可以返回整个块级作用域的返回值 (不能使用)

¶const

  • const 变量=值

  • 声明一个只读的常量 值不能改变

  • const一旦声明必须初始化(赋值)

  • 不存在变量提升(提前声明)

  • 存在暂停性死区(在变量声明前不能使用该变量)

  • 不能重复声明

  • const只是保证变量名指向的地址不变 但该地址的数据可能改变(使用const定义对象 该对象可以增删属性)

  • 将对象冻结 const 变量=Object.freeze( {} )

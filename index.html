8、对象的扩展

[TOC]

¶对象扩展

¶属性的简洁表示法

  • 属性简写 用作函数的返回值
  • 方法简写 简介写法的属性名总是字符串

¶属性名表达式

  • es5的写法

  • es6 如果属性名表达式是一个对象 会将对象转为字符串[object Object]

¶方法的name属性

  • 与函数的name属性类似
  • 如果使用了取值函数 会在方法名前加上get
  • 如果使用了存值函数 会在方法名前加上set
  • 如果对象的方法是一个symbol值 会返回这个symbol的描述

¶Object的方法

¶比较两个值是否严格相等

  • Object.is(值1,值2)

  • 不同之处 +0 不等于 -0 NaN等于自身

¶合并对象

  • Object.assign(目标对象,源对象1,源对象2) 返回复制后的目标对象

  • 将源对象的所有可枚举属性复制到目标对象上

  • 如果有同名属性后面的属性会覆盖掉前面的属性

  • 如果只有一个参数 会直接返回该参数

  • 目标对象 如果参数不是对象 会转换为对象再返回(undefined和null无法转为对象 会报错)

  • 源对象 如果不是对象 会转换为对象 转换不了就跳过(不会报错)

  • 该方法实行浅拷贝 如果源对象的某个属性的值是对象 那么目标对象拷贝得到的是这个对象的引用

¶属性的可枚举性

  • 如果可枚举性为false 某些操作会忽略当前属性
  • for in循环 Object.keys() JSON.stringify() Object.assign 会忽略不可枚举的属性
  • 所有class的原型方法都是不可枚举的

¶属性的遍历

  1. for in循环 遍历对象自身的和继承的可枚举属性(不包含symbol)
  2. Object.keys(obj) 返回一个数组 包括对象自身的(不包含继承的)所有可枚举属性(不包含symbol)
  3. Object.getOwnPropertyNames(obj) 返回一个数组 包含对象自身的所有属性（不含Symbol属性，但是包括不可枚举属性）
  4. Object.getOwnPropertySymbols返回一个数组，包含对象自身的所有Symbol属性
  5. Reflect.ownKeys返回一个数组，包含对象自身的所有属性，不管是属性名是Symbol或字符串，也不管是否可枚举
  • 遍历遵循的次序规则
    • 首先遍历所有属性名为数值的属性，按照数字排序
    • 其次遍历所有属性名为字符串的属性，按照生成时间排序
    • 最后遍历所有属性名为Symbol值的属性，按照生成时间排序

¶__proto__属性

  • 对象.__proto__ 用于读取或设置当前对象的prototype属性

  • 不建议使用

  • 建议使用Object.setPrototypeOf()（写操作）、Object.getPrototypeOf()（读操作）、Object.create()（生成操作）

¶设置一个对象的prototype对象

  • Object.setPrototypeOf(对象,原型对象)

¶读取对象的prototype对象

  • Object.getPrototypeOf(对象)

¶创建一个新对象

  • Object.create(原型对象,{属性1:{定义数据属性或访问器属性},属性2:{}})

¶遍历键值对

  • Object.values(obj) 返回值数组

  • Object.entries(obj) 返回一个包含多个数组(第一个元素键名 第二个元素值)的数组

  • Object.keys(obj) 返回键名数组

¶获取对象属性描述对象

  • es5 Object.getOwnPrototypeDescripetor(obj,'属性') 返回对象属性的描述对象
  • es6 Object.getOwnPrototypeDescriptors(obj)

7、函数的扩展

[TOC]

¶函数扩展

¶函数参数的默认值

  • function a(x,y='hello world'){}
  • 参数变量是默认声明的 不能用let或const再次声明
  • undefined会触发默认值 null不会

¶与解构赋值默认值结合使用

¶参数默认值的位置

  • 定义默认值的参数应该时函数的尾参数
  • 当前面参数设置默认值 后面参数没有设置时 前面的参数无法省略

¶函数的length属性

  • 函数.length 返回函数没有指定默认值参数的个数

  • 函数.length= 第一个设置默认值的参数的索引值

¶rest参数

  • 形式为...变量名 用于获取函数多余的参数

  • 与rest参数搭配的变量是一个数组 该变量将多余的参数放入数组中

  • rest参数必须为最后一个参数

  • 函数的length属性不包括rest参数

  • 扩展运算符可以将一个数组变成参数序列 通常在传递参数时使用

¶严格模式

  • es5 function a(b,c){ 'use strict'; // }

  • 在es6中 只要函数参数使用了默认值、解构赋值、扩展运算符 就不能显式设定为严格模式

  • 在严格模式下 对于代码有限制 但是由于先执行参数代码 在通过执行函数代码才能确定是否要执行严格模式

¶name属性

  • 函数的name属性 返回函数的函数名

  • es5中 如果将一个匿名函数赋给一个变量 会返回空字符串

  • es6 会返回实际的函数名

¶箭头函数

  • 当箭头函数代码只有一句时 会将代码执行后 返回该代码句的结果 当只有一个参数时 可以不使用 ()
  • 当有多句代码或用大括号括起后 不会自动返回 需要手动返回

¶箭头函数注意点（this等）

  • 箭头函数内的this对象是定义箭头函数时所在的对象(与上一层作用域this相同) 而不是调用时所在的对象

  • 不能作为构造函数 即不能使用new命令

  • 不能使用arguments对象 该对象在函数体内不存在 使用rest参数代替

  • 不能使用yield命令 因此箭头函数不能用作Gennrator函数

¶方法

¶转为真正数组

  • Array.from(数据结构,function(){return;},对象)

  • 将两类对象转为真正的数组 类数组的对象 可遍历的对象(Iterable) 数组返回数组

  • 扩展运算符也能将具有Iterator接口的数据结构转换为数组(类数组对象不行)

  • 任何有length属性的对象都能通过Array.from()转为数组

¶第二个参数
  • Array.from还可以接受第二个参数 作用类似于数组的map方法 用来对每个元素进行处理，将处理后的值放入返回的数组
¶第三个参数
  • 如果在map函数中使用了this 可以传入第三个参数(绑定this) 传入一个对象

¶将一组值转换为数组

  • Array.of(值1,值2,值3)

¶实例方法

¶将指定位置的成员复制到其他位置（会覆盖原有成员）

  • 实例.copyWithin(target,start,end) 三个参数都是index表示

  • target 从该位置开始替换数据

  • start 从该位置读取数据 默认为0 负值表示倒数 包括

  • end 到改位置停止读取数据 默认为length 负值表示倒数 不包括

  • 三个参数都是数值 如果不是 自动转为数值

  • 从start位置开始读取数据到end为止 将数据覆盖到从target开始的位置

¶找出第一个符合条件的数值成员

  • 实例.find(function(value,index,arr){return;},对象) 可以识别NaN

  • 直到找出第一个返回值为true的成员 然后返回改成员 否则返回undefined

  • 回调函数的参数 value 当前的成员 index 该成员对应的索引值 arr 数组本身

  • 可以接收第二个参数 绑定this

  • 传统的indexof方法不能识别NaN

¶找出第一个符合条件的数值成员的索引

  • 实例.findIndex(function(value,index,arr){return;},对象) 可以识别NaN

  • 直到找出第一个返回值为true的成员 然后返回改成员的索引值 否则返回-1

  • 回调函数的参数 value 当前的成员 index 该成员对应的索引值 arr 数组本身

  • 可以接收第二个参数 绑定this

¶使用给定值 填充数值

  • 实例.fill(value,start,end)

  • value 填充的值

  • start 起始位置 包括 默认0

  • end 结束位置 不包括 默认length

  • 将value填充到start到end的位置上(覆盖)

¶遍历键值对

¶检查数值是否包含给定的值（返回bool）es7

  • 实例.includes(数据结构,start)

  • start 表示搜索的起始位置 负值表示倒数位置 大于数组长度会重置为0

  • 与map和set数据结构中has方法的区别

    • Map结构的has方法，是用来查找键名的，比如Map.prototype.has(key)、WeakMap.prototype.has(key)、Reflect.has(target, propertyKey)。
    • Set结构的has方法，是用来查找值的，比如Set.prototype.has(value)、WeakSet.prototype.has(value)。

¶数组的空位

  • 数组的空位指数组的某一个位置没有任何值 Array(3) //[,,]
  • 空位不是undefined 空位没有任何值

¶es5对空位的处理

  • forEach() filter() every() 和some()都会跳过空位

  • map()会跳过空位 但会保留这个值

  • join()和toString()会将空位视为undefined 而undefined和null会被处理成空字符串

¶es6对于空位处理

  • es6会将空位转为undefined

  • 使用Number()方法可以将0b和0o前缀的字符串数值转为十进制

¶方法

¶检查一个数值是否有限

  • Number.isFinite(数值)

¶检查是否为NaN

  • Number.isNaN(数值)
  • 它们与传统的全局方法isFinite()和isNaN()的区别在于，传统方法先调用Number()将非数值的值转为数值，再进行判断，而这两个新方法只对数值有效，非数值一律返回false。

¶解析字符串 返回整数

  • Number.parseInt(string,radix)
  • radix 可选 要解析的数字基数(2-36之间 即进制) 如果为0或省略 以10解析
  • 如果以0x或0X开头 以16解析 如果参数小于2或大于36 返回NaN
  • 如果string为八进制字符串 返回的可能为10进制也可能为8进制

¶解析字符串 返回浮点数

  • Number.parseFloat(string)
  • 如果字符串首字符为数字(可以是空格) 对字符串解析 直到末端 不是则返回NaN
  • 将es5的方法转移到Number对象上 使语言逐步模块化

¶判断一个值是否为整数

  • Number.isInteger() 整数.0会被当成整数

¶Number.EPSILON

  • 引入一个极小的常量
  • 浮动数计算是不准确的 如果误差小于Number.EPSILON 则认为得到正确结果

¶安全整数

  • JavaScript能够准确表示的整数范围在-2^53到2^53之间（不含两个端点）超过这个范围，无法精确表示这个值

  • 引入Number.MAX_SAFE_INTEGER 和 Number.MIN_SAFE_INTEGER 表示这个范围得上下限

¶判断一个数是否在极限范围里

  • Number.isSafeInteger()

  • Math.trunc()

¶判断一个数是正数、负数、0

  • Math.sign() 正数返回 +1 负数返回 -1 0返回0 -0返回-0 其他值返回NaN

¶计算立方根

  • Math.cbrt()
